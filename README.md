# MM Pets | Exotic Companions Website
Welcome to the **MM Pets** website repository! This project is a premium, responsive portfolio website for a pet shop specializing in exotic companions like Sugar Gliders, Snakes, Marmosets, and various Cat breeds.
## üåü Project Overview
This website is designed to showcase the unique pets available at MM Pets with a luxurious and modern aesthetic. It features a dark-themed design with gold accents (`#d4af37`), responding to the client's desire for a "premium" look.
## ‚ú® Features
-   **Responsive Design**: Fully adaptable layout that works seamlessly on desktops, tablets, and mobile devices.
-   **Dynamic Hero Section**: Engaging hero area with background imagery and call-to-action buttons.
-   **Pet Gallery**:
    -   Individual cards for distinct pets (Iguana, Tegu, Snake, Marmoset, Sugar Glider).
    -   **Interactive Slideshow**: A dedicated slideshow for the Cat category (Persian, British Shorthair, Bengal) with auto-play and manual navigation.
-   **Smooth Scrolling**: Enhanced navigation experience with smooth scroll effects to different sections.
-   **Animations**: Elements reveal gracefully upon scrolling using the Intersection Observer API.
-   **Contact Form**:
    -   Integrated with **Google Sheets** via Google Apps Script for real-time form submissions.
    -   Visual feedback states (sending, sent) for better user experience.
-   **Social Integration**: Direct links to Instagram and WhatsApp for easy communication.

## üöÄ Setup & Usage
Since this is a static website, no complex build process is required.
1.  **Clone the repository**:
    ```bash
    git clone <repository-url>
    ```
2.  **Open the project**:
    Navigate to the project folder.
3.  **Run**:
    Simply open the `index.html` file in any modern web browser.
    *Tip: For the best experience during development, use a local server like "Live Server" in VS Code to avoid CORS issues with some assets or scripts.*
